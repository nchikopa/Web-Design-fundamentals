Defining an HTML Table

A table in HTML consist of table cells inside rows and columns.

Table cells

  • td stands for table data
  • Everything between opening and closing table data tags are the content of the table cell.
  • Table data elements are the data containers of the table.

Table Rows

  • tr stands for table row.
  • The number of rows and cells should be the same in each row.

Table Headers

  • th stands for table header.
  • It is used when you want cells to be headers
  • td can be used as heading.
  • By default, the text in table heading are bold and centered. This can be cahnged with CSS.
